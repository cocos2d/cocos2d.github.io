# 1.1 UI 개요


Cocos Studio의 UI를 살펴보면, 상단에는 툴바, 중앙에는 캔버스가 있습니다.<br>
그리고 캔버스의 주변에 몇 개의 패널들이 위치해 있습니다.<br>
이 인터페이스들은 View 메뉴를 통하여 켜거나 끌 수 있습니다.
