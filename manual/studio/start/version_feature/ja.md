# バージョンごとの機能

### 現在のバージョン

CocosStudioのMac版はv1.0.0.0 Betaが最新です。

バーションの定義についてはこちら: [Version definition](./../other/version/en.md)

### 現在のバージョンの機能

- Mac OS X 10.7/10.8/10.9をサポートしています。
- 冗長的なJson形式のデータを削除します。
- Windows版のデータを更新しました。

バージョン：V1.0.0.0 アルファ版

- プロジェクトの入り口を統一し,自由な編集が可能になりました。
