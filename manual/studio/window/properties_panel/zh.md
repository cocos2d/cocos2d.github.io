# 属性面板

## 简介

![](./res/properties_mian_panel.png)

属性面板是用于描述当前选中控件的属性面板，控件的主要操作都由属性框来控制，并实时展示在渲染区中。同时在其他面板对控件的操作也会实时的反应在属性框中。
