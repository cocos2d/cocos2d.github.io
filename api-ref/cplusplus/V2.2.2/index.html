About cocos2d-x

cocos2d-x open source project is designed to be a cross-platform 2D game engine for building 2D games, demos and other graphical/interactive mobile applications. It runs on OpenGL ES 1.1, and is written in C++ language, provides C++ API.
This project is based on the famous "cocos2d-iphone" project, and will keep pace with it.
